# BIDIK

MVP antarmuka BIDIK — platform pendamping siswa SMP untuk mengukur dan meningkatkan peluang masuk SMA impian.

## Menjalankan proyek

```bash
pnpm install
pnpm dev
```

Buka `http://localhost:3000`.

## Cakupan prototipe

- Landing page responsif
- Dashboard siswa
- Tiga target SMA dan peluang masuk
- Gap menuju kategori aman
- Prediksi nilai TKA
- Sprint adaptif harian
- Aktivitas terbaru

Data yang tampil masih berupa data demo. Integrasi Supabase sengaja belum mengarang tabel baru karena implementasi harus mengikuti **BIDIK Database Production V1** sebagai sumber kebenaran.
